# API Gateway Payment
A payment API gateway service built with NestJS that handles payment processing and integration.

## Description

This service acts as a payment gateway, managing payment processing and integrations with various payment providers.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn
- NestJS CLI

## Installation

```bash
npm install
```
## Commandos
npx jest
npx nest start --watch
npm test auth.controller
npm run start:dev
